# Redsoft test

## Installation:

- 
    ```sh
    $ npm install
    ```
    ```sh
    $ npm install -g npm-run
    ```

- dev (Сайт будет доступен по ссылке [localhost:9005](http://localhost:9005/).):
    ```sh
    $ npm run dev
    ```
     

- build (Сборка хранится в директории /dist):

    ```sh
    $ npm run build
    ```
